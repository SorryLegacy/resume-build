# Генератор резюме в формате DOCX

Инструмент для создания профессионального резюме в формате Microsoft Word (DOCX) из JSON файла.

## Установка

1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Для конвертации в PDF (опционально):
   - **Рекомендуется**: Установите Docker и загрузите образ LibreOffice:
     ```bash
     docker pull linuxserver/libreoffice:latest
     ```
   - **Альтернатива**: Установите LibreOffice локально (для использования с docx2pdf)
   - **Примечание**: Можно использовать другой Docker образ, установив переменную окружения:
     ```bash
     export LIBREOFFICE_DOCKER_IMAGE=your-image:tag
     ```

## Использование

1. Скопируйте пример файла и отредактируйте его:
   ```bash
   cp resume_data.example.json resume_data.json
   ```
   Затем отредактируйте `resume_data.json` со своими данными.

2. Запустите генератор:

```bash
python generate_resume.py
```

По умолчанию скрипт читает `resume_data.json` и создает `resume.docx`.

### Параметры командной строки

```bash
python generate_resume.py -i my_data.json -o my_resume.docx --pdf
```

- `-i, --input` - входной JSON файл (по умолчанию: resume_data.json)
- `-o, --output` - выходной DOCX файл (по умолчанию: resume.docx)
- `--pdf` - также создать PDF версию резюме (требуется Docker с LibreOffice или docx2pdf)

## Структура JSON файла

### personal_info
Контактная информация:
- `name` - имя и фамилия
- `email` - email адрес
- `phone` - номер телефона
- `location` - местоположение
- `linkedin` - профиль LinkedIn (опционально)
- `github` - профиль GitHub (опционально)

### summary
Краткое описание о себе (профессиональное резюме)

### experience
Массив с опытом работы. Каждый элемент содержит:
- `position` - должность
- `company` - название компании
- `period` - период работы
- `description` - описание обязанностей (строка или массив строк)

### education
Массив с образованием. Каждый элемент содержит:
- `degree` - степень/специальность
- `school` - учебное заведение
- `period` - период обучения

### skills
Массив навыков (или строка с навыками через запятую)

### additional_sections
Дополнительные разделы (языки, сертификаты, проекты и т.д.). Каждый раздел содержит:
- `title` - название раздела
- `content` - содержимое (массив строк, массив объектов {key, value}, или строка)

## Примеры

Смотрите файл `resume_data.example.json` для примера структуры и заполнения данных. Вы можете скопировать его в `resume_data.json` и заполнить своими данными.
